# ЛР‑6. Внутренняя SEO‑оптимизация веб‑приложения «ФотоТочка»

## Цель работы
Повысить корректность индексации и качество сниппетов одностраничного приложения «ФотоТочка» (маркетплейс цифровых фотографий) за счёт базовых мер внутренней SEO.

## Исходные данные
- SPA на Vite/Vue с hash-маршрутизацией, без собственного backend-SSR.
- До начала работы отсутствовали: `robots.txt`, мета description/OG/keywords, канонические ссылки; служебные разделы не закрывались от индексации.

## Ход выполнения
1) Создан `robots.txt`, в котором публичные страницы разрешены к индексации, а пути `/admin`, `/login`, `/register` и кабинеты авторов явно закрыты; дополнительно указан Host и Sitemap под будущий продовый домен (`frontend/public/robots.txt`).  
2) Обновлён основной корневой файл `frontend/index.html`: задан язык `ru`, прописан информативный `<title>`, добавлены мета `description/keywords/robots`, базовые теги Open Graph, канонический линк и `theme-color`, чтобы поисковые сниппеты были единообразными и читабельными.  
3) Настроена динамическая SEO-конфигурация маршрутов через `frontend/src/seo/seoConfig.ts` и подключение в `router.afterEach`: задана карта `title/description/robots/canonical/OG/Twitter` для ключевых страниц, служебные разделы получают `noindex`, канонический адрес формируется без `#`, чтобы исключить дубли hash-маршрутизатора.  
4) Выполнена проверочная сборка `npm run build` — ошибок не зафиксировано, метаданные попадают в итоговый `dist/index.html`.

## Результаты
- Основные страницы получают уникальные title/description/OG и канонические ссылки без хеша; служебные зоны закрыты `noindex,nofollow`.
- В корне сборки появляется актуальный `robots.txt`, ориентированный на публикацию.

## Рекомендации
- При выкладке указать реальный домен в `VITE_SITE_URL` и `robots.txt`, сгенерировать `sitemap.xml`.
- При переходе на серверный роутинг использовать `createWebHistory` и fallback на `index.html` для чистых URL.
- После появления API добавить Schema.org (ImageObject, BreadcrumbList) и OG-обложки/кнопки «поделиться» на основе реальных данных.
