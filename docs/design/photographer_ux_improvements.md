# План улучшения UI/UX для страниц фотографов

## Анализ проблем текущего дизайна

### Проблемы:
1. **Отсутствие навигации** - нет постоянной навигации между разделами
2. **Плохая информационная архитектура** - все на одной странице, нет структуры
3. **Нет контекста** - пользователь не понимает, где он находится
4. **Слабая визуальная иерархия** - все элементы выглядят одинаково важно
5. **Нет группировки** - статистика, действия и контент смешаны

## Решение: Layout с сайдбаром (как в AdminPage)

### Структура:
1. **Боковая панель навигации (Sidebar)**
   - Постоянно видимая навигация
   - Иконки и описания разделов
   - Индикация активного раздела
   - Информация о пользователе внизу

2. **Основной контент (Main Content)**
   - Top bar с заголовком и контекстом
   - Область контента с правильными отступами
   - Адаптивность для мобильных

3. **Улучшенная информационная архитектура**
   - Dashboard: обзор + быстрые действия
   - Photos: управление с улучшенными фильтрами
   - Analytics: визуализация данных
   - Settings: структурированные настройки

## Компоненты для создания:

1. **PhotographerLayout.vue** - обертка с сайдбаром
2. **PhotographerSidebar.vue** - боковая навигация
3. Переработать все страницы под новый layout

## Навигация в сайдбаре:

- 📊 Обзор (Dashboard) - `/photographer`
- 📷 Мои фотографии - `/photographer/photos`
- 📈 Аналитика - `/photographer/analytics`
- ⚙️ Настройки - `/photographer/settings`

## Визуальные улучшения:

1. **Dashboard:**
   - Крупные карточки статистики с акцентом на важные метрики
   - Группировка: Статистика → Быстрые действия → Последние фото
   - Визуальная иерархия через размеры и отступы

2. **Photos:**
   - Улучшенная панель фильтров (sticky)
   - Сетка фотографий с лучшим spacing
   - Batch actions для массовых операций

3. **Общие:**
   - Консистентные отступы и spacing
   - Правильная типографическая иерархия
   - Микро-анимации для обратной связи
