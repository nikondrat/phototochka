# Концепция главной страницы «ФотоТочка»

## 1. Цели страницы

- Показать ценность платформы и ключевые преимущества.
- Быстро провести пользователя к каталогу или регистрации.
- Продемонстрировать активность авторов (новинки, топ авторов).
- Обеспечить удобный поиск и фильтрацию прямо с главной.

## 2. Стиль и визуальный язык

- Минимализм с элементами необрутализма: четкая сетка, контрастные границы, крупная типографика.
- Палитра: `#FFFFFF` фон, `#000000` текст, `#10B981` акцент, `#059669` hover, `#ECFDF5` подложки, `#6B7280` вторичный текст, `#E5E7EB` бордеры.
- Допустимые эффекты: мягкие тени низкой высоты, аккуратные неоновые акценты для CTA (`box-shadow: 0 0 20px rgba(16,185,129,0.45)` по согласованию).
- Типографика: заголовки — `Playfair Display`, основной текст и UI — `Inter`, fallback системные шрифты.

## 3. Брейкпоинты и сетка

- Desktop ≥1280px: 12 колонок, gutter 24px.
- Tablet 768–1023px: 8 колонок, gutter 20px.
- Mobile ≤767px: 4 колонки, gutter 16px.
- Карточки изображений: базовое соотношение 3:2, для вертикальных кадров 2:3; фиксированная высота заглушек (skeleton) для предотвращения CLS.

## 4. Структура блоков (desktop)

1. **Хедер**
   - Логотип, навигация (Каталог, Блог, Помощь, О проекте), CTA `Войти` и `Стать автором`.
   - Sticky-поведение при скролле; на планшетах и мобильных — бургер-меню.
2. **Hero «Популярные фото»**
   - Левая колонка: заголовок «Создавайте. Делитесь. Вдохновляйте.», подзаголовок и CTA.
   - Правая колонка: мозаика 3–4 карточек с популярными фото, различное соотношение сторон.
3. **Поиск и фильтры**
   - Строка поиска + быстрые фильтры (категория, цвет, лицензия); на мобильных выдаётся в off-canvas.
4. **Новинки**
   - Горизонтальная карусель без «жатия»: карточки единого размера (3:2), плавная прокрутка со стрелками и затенёнными краями.
5. **Категории**
   - Плитки направлений: Природа, Животные, Люди, Город, Еда и напитки, Технологии, Абстракции.
6. **Топ авторов месяца**
   - 3–4 карточки с аватаром, кратким описанием и CTA `Смотреть профиль`.
7. **Преимущества платформы**
   - Карточки «Прозрачные лицензии», «Имитация платежей», «Безопасное хранение».
8. **CTA для авторов и покупателей**
   - Двухколоночный блок: «Начните продавать» и «Найдите идеальные фото».
9. **Футер**
   - Ссылки на блог, помощь, служебные страницы, контакты и социальные сети.

### 4.1 Мобильное меню

- Тип: полноэкранный оверлей с блокировкой скролла страницы.
- Фон: градиент `rgba(5,15,20,0.82) → rgba(5,15,20,0.72)` + blur 12px; при клике закрывает меню.
- Контейнер: карточка шириной `min(420px, 92vw)` с радиусом 24px, тенью `0 32px 60px rgba(15,23,42,0.18)`, размещается по центру с отступами `clamp(1.2rem, 7vw, 1.8rem)`.
- Верхний блок (`hero`): фон градиент `linear-gradient(135deg, #ECFDF5 0%, #FFFFFF 65%)`, логотип, теглайн «Маркетплейс цифровых фотографий», круглая кнопка `✕`.
- Основной блок: вертикальный список ссылок с разделителями, hover/active подчёркивание акцентом.
- Нижний блок: CTA `Войти` (ghost) и `Стать автором` (primary) на всю ширину.
- Анимация: fade всего меню + сдвиг карточки по Y 16px → 0 за 0.28s.
- UX: закрытие по `Esc`, по фону, по кнопке, сохранение фокуса внутри.

## 5. Секция «Новинки»

- Горизонтальный список с `scroll-snap`, карточки фиксированной ширины и стрелками для прокрутки.
- Упор на единое соотношение сторон (3:2) и аккуратные fade-переходы по краям.
- На мобильных ширина карточек адаптируется (≈78% ширины экрана), прокрутка свайпом.

## 6. Изображения и мок-данные

- Используются фотографии из библиотеки Unsplash (прямые ссылки с параметрами `auto=format&fit=crop`).
- Подборка покрывает ключевые тематики: природа, город, портреты, технологии, еда и напитки, абстракции.
- Аватары авторов также взяты с Unsplash (портретные снимки высокой четкости).

## 7. Технические заметки и открытые вопросы

- Использовать `loading="lazy"` и `srcset` для изображений с высоким разрешением.
- Выполнять prefetch первых запросов каталога при открытии страницы.
- Соблюдать контраст (WCAG AA), явно описывать состояния focus/hover.
- Компоненты Vue: `HeaderNav`, `HeroBanner`, `SearchFilters`, `PhotoCarousel`, `CategoryGrid`, `AuthorShowcase`, `BenefitsSection`, `DualCTA`, `SiteFooter`.
- Утвердить финальные тексты CTA и описаний преимуществ.
- Определить необходимость блока отзывов или кейсов.
- Решить, нужна ли подписка на рассылку на главной.
- Документ обновляется по мере развития главной страницы.
