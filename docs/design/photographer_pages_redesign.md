# План переработки дизайна страниц фотографов

## Анализ текущего стиля проекта

### Установленные паттерны дизайна:

1. **Карточки:**
   - Малые карточки: `border-radius: 16px`
   - Большие карточки: `border-radius: 24px`
   - Граница: `1px solid rgba(15, 23, 42, 0.08)`
   - Тень по умолчанию: `0 2px 8px rgba(15, 23, 42, 0.04)`
   - Тень для больших карточек: `0 18px 36px rgba(15, 23, 42, 0.08)`
   - Hover: `transform: translateY(-2px)`, `box-shadow: 0 8px 24px rgba(15, 23, 42, 0.1)`

2. **Иконки:**
   - Используются ЭМОДЗИ (📷, 👤, 👁️, 💰, 🏷️)
   - НЕ используются SVG иконки для статистики
   - Иконки в белом фоне с `border-radius: 12px`

3. **Градиенты и фоны:**
   - Градиентные фоны: `linear-gradient(135deg, rgba(16, 185, 129, 0.1), rgba(16, 185, 129, 0.05))`
   - Светлые акценты: `rgba(236, 253, 245, 0.3)`
   - Радиальные градиенты: `radial-gradient(circle at top right, #ecfdf5 0%, #ffffff 55%)`

4. **Типографика:**
   - Заголовки: `Playfair Display`, размеры `clamp(2rem, 3vw, 2.5rem)`
   - Основной текст: `Inter`
   - Eyebrow: `0.85rem`, `letter-spacing: 0.12em`, `text-transform: uppercase`

5. **Цветовая палитра:**
   - Фон: `#ffffff`
   - Текст: `#0f172a`
   - Мuted текст: `#4b5563` (var(--color-text-muted))
   - Акцент: `#10b981` (var(--color-accent))
   - Акцент темный: `#059669` (var(--color-accent-dark))
   - Поверхность: `#f5fdf9` (var(--color-surface))

6. **Spacing:**
   - Секции: `padding: 4rem 0`
   - Gap между карточками: `1.5rem`
   - Padding карточек: `1.75rem`

7. **Анимации:**
   - Transition: `0.3s ease` или `0.2s ease`
   - Hover эффекты: только `translateY(-2px)`, без масштабирования

## План переработки страниц фотографов

### 1. Dashboard (PhotographerDashboard.vue)

**Проблемы текущего дизайна:**
- Использованы SVG иконки вместо эмодзи
- Слишком яркие градиенты
- Неправильные радиусы (20px вместо 16px/24px)
- Фоновый градиент не соответствует стилю
- Слишком сложные hover эффекты

**Исправления:**
- Заменить SVG на эмодзи (📸, 👁️, ⬇️, 💰)
- Использовать градиенты как в AdminStats: `linear-gradient(135deg, rgba(16, 185, 129, 0.1), rgba(16, 185, 129, 0.05))`
- Карточки статистики: `border-radius: 16px`
- Тени: `0 2px 8px rgba(15, 23, 42, 0.04)`
- Hover: `translateY(-2px)`, `0 8px 24px rgba(15, 23, 42, 0.1)`
- Убрать фоновый градиент или использовать `radial-gradient` как в CatalogPage
- Иконки в белом фоне с `border-radius: 12px`, размер `48px`

## Принципы переработки

1. **Консистентность:** Все элементы должны использовать те же паттерны, что и AdminStats, AdminCategories, HeroBanner
2. **Минимализм:** Не добавлять лишних эффектов
3. **Эмодзи:** Использовать эмодзи для иконок, как в проекте
4. **Тени:** Только мягкие, низкой высоты
5. **Градиенты:** Только светлые, с низкой непрозрачностью
6. **Радиусы:** Строго 16px или 24px
7. **Цвета:** Только из палитры проекта
